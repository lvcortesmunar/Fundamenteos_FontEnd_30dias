🟢 DÍA 2 — CONTENIDO REAL (TEXTOS, LISTAS, IMÁGENES)
🎯 Objetivo

Agregar contenido real: texto, listas e imágenes.

🧠 Introducción

HTML responde a:

“¿Qué tipo de contenido es esto?”

Texto → <p>
Lista → <ul>
Imagen → <img>
💻 Código completo
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Dia 2</title>
</head>
<body>

<header>
  <h1>Blog simple</h1>
</header>

<main>

  <section>
    <h2>Artículo</h2>
    <p>Este es un párrafo de ejemplo.</p>
  </section>

  <section>
    <h2>Lista</h2>
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </section>

  <section>
    <h2>Imagen</h2>
    <img src="https://via.placeholder.com/300" alt="imagen">
  </section>

</main>

<footer>
  <p>Footer</p>
</footer>

</body>
</html>
