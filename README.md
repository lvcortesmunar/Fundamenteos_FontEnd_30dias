# 🔵 FASE 1 (Días 1–30) → fUNDAMENTOS FRONTEND 

**Objetivo:** ya construyes interfaces reales sin perderte.
Dominio de HTML-CSS-JS

---

# 🧩 TEMA 1 — HTML (QUÉ HAY)

## 🎯 Acción obligatoria (orden fijo)

1. Crear estructura semántica:

* header
* nav
* main
* section
* footer

2. Escribir contenido base:

* h1–h3
* p
* listas

3. Agregar navegación:

* links internos

4. Agregar elementos reales:

* imágenes
* formularios

5. Accesibilidad mínima:

* alt en imágenes
* labels en inputs

---

## 🧠 PATRÓN QUE DEBES VER

Todo HTML responde a esto:

> “¿Qué tipo de contenido es esto?”

* Texto → p
* Navegación → nav
* Sección → section
* Acción → button

Si dudas → estás sobrepensando.

---

# 🎨 TEMA 2 — CSS (CÓMO SE VE)

## 🎯 Acción obligatoria

1. Reset básico:

* box-sizing
* quitar márgenes

2. Variables:

* colores
* fuente
* spacing

3. Layout:

* primero Flexbox
* luego Grid si hace falta

4. Espaciado:

* margin / padding
* gap

5. Responsive:

* mobile first
* media queries

6. Interacciones:

* hover
* transition

---

## 🧠 PATRÓN QUE DEBES VER

Todo CSS es:

> Caja + Posición + Espacio + Color

Si algo se rompe:

* 80% es problema de **layout (flex/grid)**
* 20% es spacing

---

# ⚡ TEMA 3 — JavaScript (QUÉ HACE)

## 🎯 Acción obligatoria

1. Seleccionar elementos:

* querySelector

2. Escuchar eventos:

* click
* submit

3. Cambiar cosas:

* textContent
* classList

4. Manejar datos:

* arrays / objetos

5. Lógica:

* if
* funciones

6. Asincronía:

* fetch
* async/await

---

## 🧠 PATRÓN QUE DEBES VER

Todo JS sigue esto:

> Evento → Función → Cambio en el DOM

Ejemplo mental:

* Usuario hace click
* Se ejecuta función
* La UI cambia

Si no ves eso → estás perdido en complejidad innecesaria.

---

# 🔗 BLOQUE 4 — INTEGRACIÓN (REALIDAD)

## 🎯 Acción obligatoria

1. Conectar todo:

* HTML + CSS + JS

2. Probar:

* clicks
* formularios

3. Revisar responsive

4. Ajustar errores

5. Deploy simple



---

# 🔥 RESUMEN ULTRA SIMPLE 

Si te pierdes, vuelves a esto:

> HTML → estructura
> CSS → layout + espacio
> JS → eventos + cambios
