🟢 DÍA 1 — ESTRUCTURA BASE HTML
🎯 Objetivo

Entender cómo se construye una página web desde cero usando estructura semántica.

🧠 Introducción

Todo empieza aquí:

HTML = qué hay en la página

No diseñas, no programas.
Solo defines bloques de contenido.

💻 Código completo (copiar y ejecutar)
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Dia 1</title>
</head>
<body>

<header>
  <h1>Mi sitio web</h1>
</header>

<main>
  <section>
    <h2>Sección 1</h2>
    <p>Este es un texto de ejemplo</p>
  </section>

  <section>
    <h2>Sección 2</h2>
    <p>Otro contenido</p>
  </section>
</main>

<footer>
  <p>Footer simple</p>
</footer>

</body>
</html>

✅ Checklist (no avanzar sin esto)
 Abriste el archivo en el navegador
 Ves header, secciones y footer
 Entiendes que todo son “bloques”
